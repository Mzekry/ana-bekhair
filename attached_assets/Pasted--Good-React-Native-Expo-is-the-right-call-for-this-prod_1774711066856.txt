📄 Functional PRD — Frontend (MVP1)

🧾 TL;DR

Build a mobile app (iOS + Android) in Arabic (RTL) that allows a user to:
	•	Log in (Google + Email)
	•	Add 1 primary contact (Watcher)
	•	Tap “أنا بخير” once per night
	•	Call the watcher directly from the home screen

👉 No SMS, no backend alerts yet — just behavior validation.

⸻

🧱 Tech Stack (Frontend)
	•	React Native (Expo)
	•	Expo Router (or React Navigation)
	•	AsyncStorage (for local persistence)
	•	Expo Contacts (for contact import)
	•	Linking API (for phone calls)

Login → Onboarding → Home → Settings
